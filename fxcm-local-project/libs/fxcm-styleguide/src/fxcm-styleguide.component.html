Headings

We use classes to differentiate heading styles instead of uses elements like h1, h2, h3.
This way, a h1 element can be smaller than h2 at any time, but an accessibility reader will pick up the important targets.

Heading Main

.h-main

@include h_main();

@include h_secondary();
